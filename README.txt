***********************************************************************************************************************************************
************************************************Java Superposition T-matrix Code (JaSTA-2)*****************************************************
***********************************************************************************************************************************************

Installation Requirements:
1. Java Runtime Environment.
2. Gfortran.
3. GNU-parallel for multi-threading calculations.

Installation:
1. Download the zip file.
2. Extract the zip file in the HOME (/home/<username>) folder.

Compiling and running JaSTA-2:
1. Open a terminal in the HOME directory.
2. Enter the "JaSTA-2" folder using the command "cd JaSTA-2".
3. Type the command "./jasta" in order to run and compile JaSTA-2.

Single Wavelength option:
1. Input Parameters:
	a) Radius of Monomer
	b) Wavelength
	c) Real refractive index
	d) Imaginary refractive index
	e) Number of monomers (8, 32, 64, 128)
	f) Aggregate Type (BCCA, BPCA, User_defined)

2. Output Parameters:
	a) Scattering matrix elements
	b) Asymmetry Parameter
	c) Extinction efficiency
	d) Absorption efficiency
	e) Scattering efficiency
	f) Extinction cross-section
	g) Absorption cross-section
	h) Scattering cross-section

Multi-wavelength option:
1. Input Parameters:
	a) Radius of monomer (micron)
	b) Refractive index of the medium (mri)
	c) Dielectric file (Wavelength dependent refractive indices).
	d) Number of monomers.
	e) Aggregate type (BCCA, BPCA, user_defined),
	f) Maximum wavelength (Spline fitting option)
	g) Minimum wavelength (Spline fitting option)
	h) Step size.
	i) No. of cores or threads (Multi-threading option).

2. Output parameters:
	a) Extinction efficiency
	b) Absorption efficiency
	c) Scattering efficiency
	d) Extinction cross-section
	e) Absorption cross-section
	f) Scattering cross-section
	g) Scattering matrix elements (Scattering matrix file)
